# Twitter Clone backend

## Run locally

Requirements: Node, npm, MySQL

Create `.env` file at the root of the project with the following contents:

```javascript
    DB=<DATABASE_NAME>
    USER=<DATABASE_USER>
    PASSWORD=<DATABASE_PASSWORD>
    HOST=<DATABASE_HOST>
    CLOUDINARY_URL=<CLOUDINARY_URL>
```

`npm install`

`npm run dev`
